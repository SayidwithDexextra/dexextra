# Test 079: Collateral Accounting After a Losing Trade
# Open long at $3.00, close at $2.00. Verify collateral decreased by loss.
# loss = (3.00 - 2.00) * 50 = $50
# Deploy pre-funds each user with $1000. So starting collateral = $1000 + $5000 = $6000.
# After loss of $50, collateral should be ~$5950.
# Verifies: realized loss deducted from collateral.

STRICT ON
U1 DEP 5000
U2 DEP 5000
U3 DEP 5000

# Open at $3.00
U2 LS 3.00 1 50
U1 LB 3.00 1 50
SLEEP 500

# Close at $2.00 — loss of $50
U3 LB 2.00 1 50
U1 LS 2.00 1 50
SLEEP 500

# U1 started with $6000, lost $50. Collateral should be <= $6000
ASSERT COLLAT U1 <= 6000
