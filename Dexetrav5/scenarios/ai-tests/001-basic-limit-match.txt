# Test 001: Basic Limit Order Matching
# Two users place opposing limit orders at the same price → should fill immediately.
# Verifies: order matching, position creation, book clearing.

STRICT ON
U1 DEP 5000
U2 DEP 5000

# U1 places a limit buy, U2 places a limit sell at the same price
U1 LB 2.50 1 100
U2 LS 2.50 1 100

SLEEP 500

# Both orders should have matched — book should be empty
ASSERT BOOK_EMPTY

# U1 should be long, U2 should be short
ASSERT POSITION U1 LONG == 100
ASSERT POSITION U2 SHORT == 100
