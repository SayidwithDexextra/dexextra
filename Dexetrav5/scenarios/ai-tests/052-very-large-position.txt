# Test 052: Very Large Position (Overflow Boundary)
# Open a position with a very large ALU size to stress 18-decimal arithmetic.
# Verifies: no overflow in notional calc, entry price accuracy at scale.
# PositionManager unchecked block: totalNotional = existingNotional + newNotional.

STRICT ON
U1 DEP 500000
U2 DEP 500000

# 100,000 ALU at $2.50 = $250,000 notional
U2 LS 2.50 1 100000
U1 LB 2.50 1 100000

SLEEP 1000

ASSERT POSITION U1 LONG == 100000
ASSERT POSITION U2 SHORT == 100000
