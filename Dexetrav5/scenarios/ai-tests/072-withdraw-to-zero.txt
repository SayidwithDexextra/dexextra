# Test 072: Withdraw Entire Collateral to Zero
# Deposit then withdraw the full amount including the initial $1000 from deploy.
# Total collateral = $1000 (deploy) + $5000 (deposit) = $6000.
# Verifies: full withdrawal, no dust remaining.

STRICT ON
U1 DEP 5000

SLEEP 300
ASSERT COLLAT U1 >= 6000

U1 WDR 6000

SLEEP 300
ASSERT COLLAT U1 == 0
